# Backup antes do redesign (FadeApp)

Este backup foi criado antes de aplicar as mudanças visuais do redesign (tipografia, tema, componentes).

## Como voltar ao estado anterior

Se quiser reverter tudo, diga **"Volte ao anterior"** no chat do Cursor — o assistente restaurará os arquivos a partir desta pasta.

Ou manualmente: copie o conteúdo de cada arquivo de `_backup_before_redesign/src/` para o arquivo correspondente em `src/`, mantendo a mesma estrutura de pastas.
